⚡ LAZYEDITOR BETA

A lightning-fast, reactive terminal text editor built with Python and Textual.

Designed for developers who demand seamless performance and intuitive navigation in their coding workflow.

Note: This is a BETA release. Expect some features to be in early stages of development and potential bugs.

Features

  • Micro-Autosave: Every single typed keystroke saves straight to disk immediately.
  • Sidebar Navigation: Built-in dynamic folder tree explorer panel.
  • Integrated Terminal (BETA): Run system commands seamlessly without leaving the workspace window.
  • Tokyo Night Styling: Clean aesthetic engineered directly for developers.

Installation

Clone the beta repository using the command block below to get started:

git clone https://github.com/Janmey-Sachdev/LAZYEDITOR-BETA.git

Note: Ensure you have pip install textual installed in your local python framework prior to boot.
